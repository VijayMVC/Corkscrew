Corkscrew.SDK.providers.database Namespace
This namespace contains provider implementations for backend database systems that can be used by Corkscrew.
Classes
  Class Description
CSDatabaseProviderFactory
This class determines the backend provider class from the list of loaded or attached assemblies and provides the correct one based on the connection string. The CSDatabaseProviderFactory class is what helps Corkscrew support multiple (database) backend engines to store its data.
CSMySqlProvider
Provider for MySQL database engine
CSSqlServerProvider
Class implementing the SqlServer database provider
