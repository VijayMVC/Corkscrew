Properties
  Name Description
AuthenticatedUser
Returns the authenticated user credential using which this item was created or opened.
Created
Gets the date/time of creation
CreatedBy
Gets the user who created the entry.
Farm
Gets reference to the Farm
Filename
Get/set filename
FilenameExtension
Get/set filename extension
FilenameWithExtension
Returns the filename with extension.
FullPath
Get full site-rooted path to this item
History
Returns all the change history for this filesystem entry
Id
Guid of the entry
IsArchive
Get/set if item is archive ready. Not used by Corkscrew system.
IsFolder
Gets if is a folder. Cannot be changed.
IsHidden
Get/set if is a hidden item. Hidden items are not rendered to UI, but can be accessed programmatically.
IsReadonly
Get/set if item is readonly. Content of readonly files cannot be changed. If is a directory, no new files can be added or removed.
IsRootDirectoryEntry
Returns if this is the RootDirectory entry
IsSystem
Gets if is a system item. Cannot be set through business layer. Can only be modified by Features during (un)installation.
ItemLocation
Location of the filesystem item. This can be either ConfigDB or SiteDB.
LastAccessed
Gets the date/time of last access. Will always be equal to current time.
LastAccessedBy
Gets the user who last accessed the entry. Will typically be equal to current user.
Modified
Gets the date/time of last modification
ModifiedBy
Gets the user who last modified the entry
ParentDirectory
Reference to the Directory containing this entry
ParentDirectoryPath
Get/set full site-rooted path to the parent directory
RootDirectory
Returns the root directory of the Site
Site
Reference to the CSSite this entry belongs to
