CSSite Properties

The CSSite type exposes the following members.

Properties
  Name Description
AllConfiguration
Collection of all the configuration information. This call is the same as Farm.AllConfiguration NOTE: Configuration is always global and not specific to a particular site. Feature configuration is not accessed through this method.
AllContentTypes
Collection of all the configured content types. This call is the same as Farm.AllContentTypes NOTE: This is always global and not specific to a particular site. Key = File Extension, Value = Content type MIME name
AllSiteAdministrators
Gets a collection of all administrators for this Site. Will also include Farm admins, since they are also site admins. For Farm Admins, it will return all admins. For non-administrator users, will return empty list.
AllUsers
Gets a collection of all the users in the Farm. Will also include global, anonymous and other grants. For Farm Admins, it will return all users, otherwise will return only authenticated user. This call is the same as Farm.AllUsers
AllWorkflowAssociations
Collection of workflows associated to this site.
AuthenticatedUser
Returns the user account used to instantiate this site object.
Columns
Table column definitions defined at this site
ContentDatabaseName
Get the name of the SiteDB database for this Site. If no explicit name has been set, it will default to the farm database name.
ContentDatabaseServerName
Get the name of the server hosting the content database.
Created
Date and time of creation. Set by constructor.
CreatedBy
User who created. Set by constructor.
Description
Description of the site
DNSNames
Manage DNS name mappings for this Site. Cannot manage DNS names for ConfigDB site.
Farm
Reference to the parent farm
FullPath
Gets the Corkscrew Uri to the site
History
Returns all the change history for this site
Id
Unique Id of the site
IsAuthenticatedUserFarmAdministrator
Returns if the authenticated user is a farm administrator
IsAuthenticatedUserSiteAdministrator
Returns if the authenticated user is a Site administrator
IsConfigSite
Returns if the site is the Config site
IsSiteOtherThanConfigSite
Returns if the site is other than the Config site
Modified
Date and time of modification. Set internally or by persistence.
ModifiedBy
User who modified. Set internally or by persistence.
Name
User friendly name of the site
QuotaBytes
Set the site quota, in bytes. A value of zero disables quota.
RootFolder
Gets the root folder for the site
Tables
Gets a collection of all tables in the site
