aquariusos.corkscrew.library
Corkscrew CMS is a enterprise-class content management system. It features a multi-site architecture and allows management of content (documents, media, etc) via .NET code.
Namespaces
Namespace Description
Corkscrew.SDK.ActiveDirectory
Contains classes and functionality that enables Corkscrew applications interact with Windows Active Directory domains.
Corkscrew.SDK.constants
This namespace contains various constants and enum values used in Corkscrew
Corkscrew.SDK.diagnostics
This namespace contains functionality for writing logging and tracing messages.
Corkscrew.SDK.exceptions
This namespace contains methods for dealing with exceptions and throwing them
Corkscrew.SDK.objects
This namespace contains the object hierarchy of the Corkscrew CMS system
Corkscrew.SDK.providers.database
This namespace contains provider implementations for backend database systems that can be used by Corkscrew.
Corkscrew.SDK.providers.filesystem
This namespace contains virtual filesystem providers. These providers are used in a web context to provide transparent access to content hosted in Corkscrew.
Corkscrew.SDK.providers.httpmodules
This namespace contains custom HTTP Modules that modify the behaviour of responses and requests in a Corkscrew context. These can only be used by Websites accessing Corkscrew hosted content.
Corkscrew.SDK.security
This namespace contains functionality for User and Permission (ACL) management.
Corkscrew.SDK.tools
This namespace contains helper and auxilliary functionality common to other rmodules in Corkscrew codebase.
Corkscrew.SDK.workflow
This namespace contains functionality to create, manage and execute Workflows in the Corkscrew context.
