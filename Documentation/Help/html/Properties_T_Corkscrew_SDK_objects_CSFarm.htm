CSFarm Properties

The CSFarm type exposes the following members.

Properties
  Name Description
AllAdministrators
Get all Farm admins. Will not contain system users. For Farm Admins, it will return all admins, otherwise will return empty list!
AllConfiguration
Collection of configuration values. Writable to Farm Admins. NOTE: Configuration is always global and not specific to a particular site.
AllContentTypes
Collection of registered Content Types (MIME types). Writable to Farm Admins. NOTE: This is always global and not specific to a particular site. Key = File Extension, Value = Content type MIME name
AllSites
All the sites in this farm. Writable only to Farm Admin users. Note, only the sites that the user (used in Open[]) has access to will be returned!
AllUserGroups
Gets all the user groups in the farm.
AllUsers
Get all users in the farm. Will not include built-in users. For Farm Admins, it will return all users, otherwise will return only authenticated user!
AllWorkflowAssociations
Collection of workflows associated at the farm level.
AllWorkflowDefinitions
Collection of workflows defined in this farm
AuthenticatedUser
Returns the user account used to instantiate this site object. This should *never* be null because there is no way to get a CSFarm without this being set!
ConfigDatabaseServerName
Name of the server hosting this farm's ConfigDB
DefaultSite
Returns the default site (Config Site)
FullPath
Returns the full path to this Farm
Id
Returns the farm Id.
IsAuthenticatedUserFarmAdministrator
Returns if the AuthenticatedUser is a farm administrator
IsWorkflowEnabled
Returns if workflows are enabled for the farm
RootDirectory
Returns the root directory of the Configuration site.
