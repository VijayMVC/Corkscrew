Corkscrew.SDK.providers.httpmodules Namespace
This namespace contains custom HTTP Modules that modify the behaviour of responses and requests in a Corkscrew context. These can only be used by Websites accessing Corkscrew hosted content.
Classes
  Class Description
CorkscrewHttpResponseModule
Handles HTTP Response modification We change the headers in the response data during PreSendRequestHeaders event in the pipeline. We change the ASP.NET "Server" header to read our name, and modify the Content-Type header in some cases.
CorkscrewMapRequestHandlerModule
This module takes care of mapping requests for static files to the IIS StaticFileHandler module. Without this module, this mapping will need to be taken care of in the web.config [handlers] node.
